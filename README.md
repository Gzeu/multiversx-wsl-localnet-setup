## 🚀 Quick Start - Enterprise Setup

### ⚡ Ultra-Fast Enterprise Deployment
```bash
# Clone and initialize enterprise environment
git clone https://github.com/Gzeu/multiversx-wsl-localnet-setup.git
cd multiversx-wsl-localnet-setup

# Complete enterprise setup in one command
make enterprise-setup
```

### 🎯 Setup Options
```bash
# Choose your deployment level:
make quick-start           # Basic development (2 minutes)
make enterprise-setup      # Enterprise infrastructure (5 minutes)
make full-enterprise       # Complete suite with AI & mobile (10 minutes)
make wizard               # Interactive guided setup
```

### 🌐 Access Enterprise Services
After setup, access your enterprise environment:
- **🎯 Localnet API**: http://localhost:7950
- **📊 Grafana Dashboards**: http://localhost:3000 (admin/admin)
- **📈 Prometheus Metrics**: http://localhost:9090
- **🎮 Node Exporter**: http://localhost:9100
- **🚨 Alert Manager**: http://localhost:9093
